# neural-net-from-scratch

## Tagline
A simple implementation of a neural network from scratch in Python.

## What/Why
This project provides a basic implementation of a neural network in Python, allowing users to perform forward and backward passes, as well as optimization. It is designed to be a starting point for understanding neural networks and their implementation.

## Install
To install the project, run the following command:
```bash
pip install numpy
python setup.py install
```
## Usage
To use the neural network, create an instance of the `NeuralNetwork` class and provide the necessary input and output layers. You can then perform forward and backward passes using the `forward_pass` and `backward_pass` methods.

## Build from Source
To build the project from source, navigate to the project directory and run the following command:
```bash
python setup.py build
```
## Project Structure
The project is organized into the following directories:

* `src/`: Contains the main implementation of the neural network.
* `tests/`: Contains unit tests for the project.

The project includes unit tests for the main module, neural network module, activation functions module, and optimizers module. To run the tests, navigate to the project directory and run the following command:
```bash
python -m unittest discover
```
